# Product previews

Drop the homepage screenshots here, named after the product slug :

| File                       | Product           |
| -------------------------- | ----------------- |
| `creneau.png`              | Créneau           |
| `nurapic.png`              | Nurapic           |
| `sama-reservation.png`     | Sama Reservation  |
| `ticket-events.png`        | Ticket-Events     |

PNG, JPG and WebP all work — the `ProductPreview` component picks them up
via `next/image`.

**Recommended dimensions** : 1600 × 1000 px (16:10), under 300 KB once
compressed. WebP gives the best quality-to-weight ratio.
